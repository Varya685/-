Сайт - Киноафиша
Задача проекта: научиться получать данные с сайта (парсинг).
Разработать парсер для сервиса Киноафиша, который возвращает информацию о новостях.
В процессе парсинга были добавлены новые библиотеки: requests, BeautifulSoup, pandas.
Автоматизация сбора данных снизит трудозатраты на ручной ввод и повысит актуальность информации.
Основная цель: создать инструмент для автоматического сбора данных.
Процесс парсинга:
- отправка HTTP‑запроса к странице Киноафиша;
- извлечение данных: название новости, дата;
- обработка данных.
Выходные данные: таблица с данными.
